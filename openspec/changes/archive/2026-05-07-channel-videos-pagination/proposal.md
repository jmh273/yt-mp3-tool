# Proposal: 單一頻道全歷史影片分頁載入 (Channel Videos Pagination)

## 目標 (Objective)
允許使用者點擊左側單一訂閱頻道時，在右側面板列出該頻道「所有」歷史影片，由新到舊排列。為避免 API Quota 消耗過快及維持效能，採用分頁機制，每次載入 50 部影片。

## 動機 (Motivation)
目前工具僅能透過 RSS 取得各頻道最新約 15 部影片，無法瀏覽或下載頻道較早期的內容。若想下載舊影片，使用者必須手動尋找並貼上 YouTube 連結，相當不便。

## 成功標準 (Success Criteria)
1. 點擊頻道時，右側動態牆切換為該頻道的專屬視圖。
2. 初始載入該頻道最新的 50 部影片。
3. 列表底部提供「載入更多」按鈕，點擊後再載入次 50 部影片，直到無影片為止。
4. 抓取過程不使用耗費配額極高的 `search.list` API，而是使用高效率的 `playlistItems.list` API。
