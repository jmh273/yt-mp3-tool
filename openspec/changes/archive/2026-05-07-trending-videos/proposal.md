# Proposal: 台灣地區發燒影片 (Trending Videos)

## 目標 (Objective)
在應用程式左側欄新增一個「發燒影片」按鈕。點擊後，右側動態牆會顯示目前 YouTube 台灣地區最熱門的 50 部影片，讓使用者除了自己訂閱的頻道外，也能探索大眾熱門內容。

## 動機 (Motivation)
YouTube 官方 API 不支援獲取個人的首頁推薦，但為了增加探索新內容的功能，我們選擇實作成本最低且最穩定的方案：透過官方 API 取得當地的「發燒影片 (Trending)」。

## 成功標準 (Success Criteria)
1. 介面左欄「最新影片」下方出現「發燒影片」按鈕。
2. 點擊後，中欄切換為顯示台灣地區目前的發燒影片（至多 50 部）。
3. 這些發燒影片一樣會受到「設定」中的「影片長度過濾」限制（例如只顯示 3~60 分鐘的影片，過濾掉太短的 Shorts 或太長的直播）。
4. 點擊這些影片的 checkbox 同樣可以加入下載佇列。
